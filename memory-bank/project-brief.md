# Project Brief — Claw'd Forge v2

## Scope
Claw'd Forge is a standalone Electron desktop app that wraps the Claude CLI workflow skill in a visual dashboard. v2 replaces the terminal-centric v1 with an iOS-friendly interactive dashboard where users never touch a terminal.

## Goals
1. Polished, approachable dashboard — iOS-level clarity
2. Interactive during presearch — structured cards, not terminal typing
3. Informative during build — progress at a glance, detail on demand
4. Personality-driven — Claw'd mascot gives the process a face
5. Tied to real execution — not a mock or demo

- 29 Must-have, 3 Should-have
- See PRESEARCH.md Requirements Registry for full list
- MVP: All R-IDs except R-007, R-020, R-030

## Target User
Single developer using Claude CLI workflow skill. Wants visual feedback without terminal complexity.
